# Firebase Auth with Expo React Native

## Starter project for using:
- React Native
- Expo
- Firebase

## Setup:

- Open `constants/ApiKeys.js` file and update these with your actual firebase config values:
```
export default {
  FirebaseConfig: {
    apiKey:      "",
    authDomain:  "",
    databaseURL: "",
    projectId: "",
    storageBucket: "",
    messagingSenderId: ""
  }
}
```
- Run `npm install`
